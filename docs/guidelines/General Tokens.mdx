# General Tokens

General tokens are predefined values that can be applied to various elements in the design system. These tokens ensure consistency and make it easy to apply styles across the application.

## Border

Borders are used to define the edges of elements and can help to create separation and emphasis. We use tokens to standardize border radius and border width across the application.

### Border Radius

The border radius tokens define the rounding of the corners of an element.

- **None**: 0
- **Small**: 4px
- **Medium**: 8px
- **Large**: 16px
- **Pill**: 9999px

### Border Width

The border width tokens define the thickness of the borders of an element.

- **None**: 0
- **Small**: 1px
- **Medium**: 2px
- **Large**: 4px

## Opacity

The opacity tokens define the transparency level of an element.

- **None**: 0
- **Low**: 0.2
- **Medium**: 0.5
- **High**: 0.8
- **Full**: 1

## Z-Index

The z-index tokens define the stack order of elements.

- **Sticky**: 1000
- **Fixed**: 1010
- **Modal Backdrop**: 1020
- **Modal**: 1030
- **Popover**: 1040
- **Dropdown**: 1050
- **Toast**: 1060
- **Tooltip**: 1070
- **Loading**: 1080
