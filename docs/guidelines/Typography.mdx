# Typography

Typography is a key element of design that helps to create a visual hierarchy and improve readability. This guide provides an overview of the typography styles used in our design system.

## Font Family

Our design system uses the following font family:

- Lato, sans-serif

## Font Weights

We use a variety of font weights to create emphasis and hierarchy in our text. The available font weights are:

- Regular: 400
- Medium: 500
- Bold: 700
- Black: 900

## Font Tokens

Font tokens are predefined styles that can be applied to text elements. These tokens ensure consistency and make it easy to apply typography styles across the application.

Usage example:

```css
// default usage
@include $body-1;

// custom example
@include $body-1($responsive: true, $breakpoint: 'md');
```
