## Icon Sizes & Tokens

Icon tokens are predefined styles that can be applied to icons. These tokens ensure consistency and make it easy to apply icon styles across the application. Weight and breakpoint for responsive sizes can be adjusted as needed.

Usage example:

```css
// default usage
@include $icon-1;

// custom example
@include $icon-1($responsive: true, $breakpoint: 'md', $weight: 'bold');
```
