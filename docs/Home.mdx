# Welcome to **Sanhauá**

**Sanhauá** is a design system created to facilitate the development of user interfaces in a consistent and efficient manner. It consists of a set of components, styles, and guidelines that can be used to create high-quality user interfaces.

Visit the repository on [GitHub](https://github.com/fpcoutinho/sanhaua).
