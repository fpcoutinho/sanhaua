# Bem-vindo ao **Sanhauá**

O **Sanhauá** é um _design system_ criado para facilitar a criação de interfaces de usuário de forma consistente e eficiente. Ele é composto por um conjunto de componentes, estilos e diretrizes que podem ser utilizados para criar interfaces de usuário de alta qualidade.

Visite o repositório no [github](https://github.com/fpcoutinho/sanhaua).
